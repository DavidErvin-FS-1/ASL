# Exploring other languages with Docker

This repository contains a collection of Dockerfiles for exploring other programming languages. Inside the main [exploring_docker](./exploring_docker) directory, each language has its own directory, and each directory contains a Dockerfile and associated files for building and running the Docker image.

## Languages

- [C++](./exploring_docker/cpp)
- [Go](./exploring_docker/go)
- [Java](./exploring_docker/java)
- [Lua](./exploring_docker/lua)
- [Node.js](./exploring_docker/node)
- [Perl](./exploring_docker/perl)
- [PHP](./exploring_docker/php)
- [Python](./exploring_docker/python)
- [Ruby](./exploring_docker/ruby)
- [Rust](./exploring_docker/rust)
